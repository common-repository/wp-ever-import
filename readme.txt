This plugin lets you import xml/csv and map the fields to generate respective posts and pages

== Description ==

-Lets you import xml or csv files to generate respective posts/pages in WordPress website.
-Preview and choose among different data sets in an xml file.
-Map fields using drag n drop.
-Supports feature Image, Categories and Tags.
-Flexibility in choosing batch size for running imports from big files.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the EverImport->Imports from the side menu in the dashboard to manage your imports.
